Ugurlabs Agents

// design mockups · v0.1 · for Electron build reference

Click-through HTML mockups for the Ugurlabs Agents desktop app. Each screen below is a standalone HTML file using shared design tokens in _design.css. Use as the visual spec when building the Electron app — the CSS variables, typography scale, spacing, and component patterns translate directly to React/Tailwind/etc.

How to use these: open them in a browser at desktop window size (~1280×800 minimum). All screens share _design.css — change a token there and every screen updates. When porting to Electron + React, treat the CSS variables as your theme tokens, the component patterns (cards, buttons, status dots, pills, tags) as your component library spec, and the layouts as the source of truth for spacing and density.
